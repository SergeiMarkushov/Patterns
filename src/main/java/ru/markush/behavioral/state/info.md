<h1>State // Состояние</h1>
<hr>
<h3>
Позволяет объекту варьировать свое поведение в зависимости от внутреннего состояния. Извне
создается впечатление, что изменился класс объекта.<br><p></p>
Паттерн Состояние предполагает выделение базового класса или интерфейса для всех допустимых
операций и наследника для каждого возможного состояния.
</h3>
