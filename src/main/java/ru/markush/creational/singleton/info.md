<h2>Гарантирует, что у класса есть только один экземпляр, и предоставляет к нему глобальную
точку доступа.
Класс Одиночка объявляет свой конструктор приватным, что, с одной стороны, не позволяет
клиентскому коду самовольно порождать инстансы Одиночки, с другой – предоставляет статический
метод getInstance, возвращающий инстанс объекта Одиночки, инстанциация которого полностью
контролируется классом Одиночкой.</h2>
